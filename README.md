# Customer-Segmentation-and-Churn-Pattern-Analytics-in-European-Banking
In banking: a customer closes their account, stops using services, moves to a competitor. If you have 100 customers and 20 leave this year → 20% churn rate. That's it. The expensive part is that acquiring a new customer costs 5–7x more than keeping an existing one. This Project is all about locating and quantifying churn using a EU Data.
